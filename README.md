# PhoneMart - Smartphone Store SPA

## 📌 Description
PhoneMart is a Single Page Application built using Vue 3, TypeScript, and Tailwind CSS. It displays smartphone products using the DummyJSON API.

## 🚀 Features
- View smartphone products
- Search functionality
- Product detail modal
- Responsive UI

## 🛠️ Technologies
- Vue 3
- TypeScript
- Tailwind CSS
- Vite

## 🔗 API
https://dummyjson.com/products/category/smartphones
